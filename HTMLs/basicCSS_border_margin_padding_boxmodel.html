CSS Box Model

  1. Content: The actual text, image, or data inside the box.
  2. Padding: Space between the content and the border.
  3. Border: Surrounds the padding (optional).
  4. Margin: Space between the element and its surroundings.
  5. Width/Height: Defines the size of the content area (can use %, px, etc.).
